## Contribution guide

1. Clone the project
```bash
  git clone https://github.com/Mesi21/Code-To-Do.git
  cd Code-To-Do
```

To start live-server

```bash
    npm start
```
To webpack watch

```bash
    npm run watch
``` 

2. Then :heavy_plus_sign: add your contribution
3. And Feel free to make a PR
